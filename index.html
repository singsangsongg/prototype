CLI Agent 모델

SaaS를 버리고 고객 환경에 install되는 CLI로 전환. 고객이 본인 AI 도구(Claude Code 등)에서 자연어로 운영. Self-Serve ($10K–$50K)와 Managed ($50K–$100K+) 두 tier.

전략 메모 v2 (PDF · 18분) Prototype v2 · CLI Install Prototype v2 · 9 Agents Prototype v2 · 가격 비교
두 버전을 다 보는 이유: v1과 v2 사이에 큰 가정 변화가 있었습니다. v1은 우리가 SaaS dashboard를 만들고 고객이 거기 로그인한다고 가정했는데, Bolta CLI 사례를 본 뒤 그 가정 자체를 버렸습니다. v2는 고객의 AI 환경에 install되는 agent 모델로 재구성한 결과입니다. 회의에서 두 버전을 비교해보면 "왜 v2로 가야 하는가"가 명확해집니다.

회의 진행 방식: Amazon 스타일 6-pager 메모입니다. 회의 시작 후 처음 10–15분은 v2 메모를 함께 읽고, 그 다음 메모 6번 섹션(같이 정해야 할 것들)의 Q1–Q5를 토론합니다. v1은 참고용으로 옆에 두세요.
