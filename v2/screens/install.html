우리 dashboard에 로그인할 필요 없습니다. 본인이 쓰는 AI 도구(Claude Code · Cursor · GPT)에 우리 agent를 install해서 자연어로 retail media를 운영하세요. 데이터는 본인 환경에서, AI 모델은 본인이 선택, 토큰 비용은 본인 계정으로.

1
CLI 설치
터미널에서 한 줄. 환경에 따라 npm·brew·curl 중 편한 방법을 고르세요.
# npm
$ npm install -g @retail-media/cli

# Homebrew (macOS)
$ brew install retail-media/tap/retail

# curl (Linux/macOS)
$ curl -fsSL https://retail-media.app/install.sh | sh
2
API Key 발급 + 인증
회원가입 후 개발자 페이지에서 API key를 받아 CLI에 등록합니다. Amazon Ads · DSP · AMC 권한이 한 번에 연결됩니다.
$ retail configure --api-key YOUR_API_KEY
✓ Amazon Ads connected (account: A1B2C3...)
✓ Amazon DSP connected
✓ AMC connected
✓ Ready to run agents
3
본인 AI 도구에서 자연어로 명령
설정 끝. 이제 Claude Code · Cursor 등에서 자연어로 retail media를 운영합니다. CLAUDE.md와 llms.txt가 자동 내장되어 AI 도구가 명령을 정확히 해석합니다.
Claude Code · Brand A 프로젝트
YOU
지난주 경쟁사 ASIN 5개 BSR 변화 보여줘
CLAUDE CODE
retail competitor monitor --asins ABC,DEF,... --range 7d를 실행합니다…

결과: 5개 ASIN 중 3개가 카테고리 상위 100위 진입. 가장 큰 변화는 ASIN-XYZ가 BSR 234→89위 (155위 상승). 가격을 $34.99 → $28.99로 인하한 것이 직접 원인으로 보입니다. 전체 리포트는 ~/retail-reports/competitor-2026-05-22.md에 저장했습니다.
YOU
우리 Brand A의 다음 달 DSP plan 짜줘. Awareness 60%, Conversion 40%.
CLAUDE CODE
retail dsp campaign-design --brand brand-a --awareness 0.6 --conversion 0.4를 실행합니다…

$30K 광고비 가정. Awareness $18K — Prime Video $11K + Twitch $5K + Fire TV $2K. Conversion $12K — Cart abandoner $7K + subscriber lapsed $5K. 캠페인 구조와 audience 설정 초안을 보여드릴까요?
왜 CLI인가: 우리는 SaaS dashboard를 만들지 않습니다. 본인이 이미 쓰는 AI 도구가 곧 인터페이스입니다. 우리 서버는 광고 API 인증·게이트웨이만 담당하므로, 광고 데이터는 본인 환경에서 처리됩니다. AI 모델도 본인이 선택(Claude·GPT·Cursor 무관), 토큰 비용도 본인이 부담합니다.
