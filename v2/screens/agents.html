9개 Agent — 우리 도메인 지식의 응축 v2

각 agent는 단순한 API wrapper가 아닙니다. 우리가 직접 K-Brand를 운영하면서 쌓은 retail media 도메인 지식 — bid·budget 최적화 로직, 인플루언서 매칭 풀, AMC 질의 라이브러리, DSP funnel 분해 framework — 이 코드로 응축된 것입니다. CLI는 이것을 본인 AI 도구로 전달하는 그릇일 뿐입니다.

왜 도메인 지식이 본질인가: AI 모델은 누구나 갖다 쓸 수 있습니다. CLI 패키징도 누구나 만들 수 있습니다. 시장에서 우리 agent가 의미 있는 이유는 그 안에 응축된 운영 판단·benchmark·playbook 때문입니다. 같은 명령을 일반 Claude에 시키면 "DSP가 무엇인지 설명하는" 일반론이 나옵니다. 우리 agent에 시키면 한국 → 미국 K-Brand 운영 경험에서 검증된 판단이 나옵니다. "agent의 코드 자체보다, 그 코드가 담고 있는 운영 판단·benchmark·playbook이 본질이다." — 전략 메모 v2, Section 2
01 Intelligence 시장과 내 위치 파악 — 진단의 기준선이 되는 데이터를 만든다
경쟁사 모니터
응축된 도메인 지식

BSR 변화 임계치 · 가격 변동 패턴 · 리뷰 velocity 정상 범위

CLI
$ retail competitor monitor --asin ABC --range 7d

경쟁사 ABC의 BSR·가격·리뷰 변화 일간 추적해줘

카테고리 트래커
응축된 도메인 지식

CLI
$ retail category track --category beauty

내 카테고리 순위 증감과 신규 진입자 알림 설정해줘

My BSR 트래커
응축된 도메인 지식

광고 의존도 진단 · 유기 vs 광고 트래픽 정상 비율 · keyword 회복 패턴

CLI

내 ASIN 5개의 BSR · 유기 트래픽 · keyword 추이 보여줘

02 Demand Creation 외부에서 인지·트래픽 생성 — Amazon 안에서만 풀 수 없는 갭을 해결
인플루언서 매칭
응축된 도메인 지식

TikTok·Meta 시딩 풀 (~2,400) · 카테고리별 매칭 패턴 · 후행지표 검증 프레임

CLI
$ retail influencer match --followers 10K-50K

팔로워 10K–50K, beauty, 미국 거주 시딩 후보 30명 추천해줘

DSP Audience Builder
응축된 도메인 지식

ASIN/category viewer 효율 비교 · lookalike 시드 선정 기준 · audience 중첩 회피

CLI
$ retail dsp audience --asin ABC --type lookalike

ASIN viewer + category viewer lookalike 자동 생성해줘

DSP Campaign Designer
응축된 도메인 지식

Awareness/Consideration/Conversion 단계별 인벤토리 (Prime Video·Twitch·Fire TV) 효율 · 예산 배분 framework

CLI
$ retail dsp design --budget 10K --awareness 0.6

광고비 $10K로 Awareness 60% / Consideration 30% / Conversion 10% plan 짜줘

03 Demand Capture Amazon 내부 효율 — Sponsored Ads와 AMC 분석으로 전환 끌어올리기
Sponsored Campaign Designer
응축된 도메인 지식

캠페인 구조 (SP/SB/SD) 조합 patterns · 신규 ASIN 런칭 플레이북 · cannibalization 회피

CLI
$ retail sp design --launch --asins NEW1,NEW2,NEW3

신규 ASIN 3개 런칭 캠페인 구조 설계해줘

Sponsored RTB
응축된 도메인 지식

Bid·budget·placement 최적화 로직 · time-of-day 패턴 · keyword 회수율 모델

CLI
$ retail sp rtb --auto --range 7d

지난주 keyword 성과 기반으로 bid·budget 재조정해줘

AMC Insight
응축된 도메인 지식

자주 묻는 audience·도달·전환 질의 라이브러리 · 자연어 → SQL 매핑 · 도달 미달 진단 패턴

CLI
$ retail amc query --type reach-gap --audience cart

Cart abandoner audience의 retargeting 도달 미달 분석해줘

9개라는 숫자에 집착하지 않습니다. 도메인을 운영하면서 패턴이 충분히 반복된 agent만 만듭니다. 효과가 약한 agent는 deprecate합니다. 새 agent는 한국 → 미국 K-Brand 운영 중에 자연스럽게 발견됩니다. 카탈로그 자체가 우리 도메인 지식의 현재 상태를 반영합니다.
